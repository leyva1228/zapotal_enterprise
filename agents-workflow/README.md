# Agents Workflow

Este directorio contiene el workflow operativo para agentes sobre Zapotal Enterprise.

## Objetivo

Dar un lugar unico y predecible para:

- auditorias
- planes de implementacion
- tareas de ejecucion
- revisiones de verificacion
- auditorias post-implementacion

## Tecnologias cubiertas

- `comunidad_zapotal_backend/`
- `comunidad_zapotal_frontend/`
- `comunidad_zapotal_mobilebff/`
- `comunidad_zapotal_mobile/`

## Carpeta compartida

`shared/` contiene:

- politicas
- templates
- checklists
- referencias movidas desde `docs/` que sirven al workflow
