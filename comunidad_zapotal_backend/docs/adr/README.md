## Formato

Cada ADR sigue el formato:
- **Contexto**: Situación que motiva la decisión
- **Decisión**: Qué decidimos
- **Consecuencias**: Trade-offs y efectos
- **Alternativas consideradas**: Otras opciones evaluadas
